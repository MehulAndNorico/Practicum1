Voorbeelden van Social Plugins

Javascript SDK

Om social plugins te kunnen gebruiken, heb je de Javascript SDK nodig. Die initialiseer je als volgt:

<script>(function(d, s, id) {
var js, fjs = d.getElementsByTagName(s)[0];
if (d.getElementById(id)) return;
js = d.createElement(s); js.id = id;
js.src = "//connect.facebook.net/nl_NL/sdk.js#xfbml=1&appId=APPID&version=v2.0";
fjs.parentNode.insertBefore(js, fjs); }(document, 'script', 'facebook-jssdk'));</script>

Op de plek van APPID moet je een token invullen die je kunt aanvragen op !!!NOG AANPASSEN!!!

class="fb-like"
data-href="http://www.cs.uu.nl/docs/vakken/b2wt/"
data-layout="standard"
data-action="like"
data-show-faces="true"
data-share="true"
></div>

<div
class="fb-like"
data-href="http://www.cs.uu.nl/education/"
data-layout="box_count"
data-action="like"
data-show-faces="false"
data-share="false"
></div>

Daarnaast zijn er nog twee layouts: button_count en button.
