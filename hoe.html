API

API, staat voor Application Program Interface, is een set van routines, protocollen en tools die gebruikt kunnen worden om software applicaties te bouwen. Een API specificeert hoe software componenten met elkaar moeten communiceren en worden gebruikt tijdens het programmeren van graphical user interface (GUI) componenten.[1] Een API assisteert programmeurs bij het schrijven van code voor de interactie met andere software. Deze interacties kunnen varieren van de low-level tools die Windows en OS X applicaties permissie geven om bepaalde aspecten van het onderliggende operating system te gebruiken, tot de apps op je telefoon. Kortom, een API defineert een manier waarop comuter programma’s met elkaar kunnen communiceren.[2] Er zijn vele verschillende vormen van API. In dit verslag wordt er dieper ingegaan in de API’s van verschillende sociale media platforms. Ten eerste bespreken we hoe een Web API in elkaar zit en daarna worden de API van Facebook en Twitter doorgenomen.
