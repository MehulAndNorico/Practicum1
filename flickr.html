Flickr AP

De Flickr API[1] is gevarieerd in formats en SDK’s.

Requests kunnen gedaan worden met REST, XML-RPC en SOAP. Rest werkt via een URL, waardoor je met HTTP GET en POST kan interacteren met de API. XML-RPC en SOAP werken met XML bestandjes met een specifieke opbouw. De response kun je krijgen in REST, XML-RPC, SOAP, JSON en PHP.

Voor Flickr zijn er veel SDK’s, maar deze zijn niet officieel. Flickr noemt ze zelf API Kits. Er zijn SDK’s voor de volgende platformen:

  • ActionScript
  • C
  • Cold Fusion
  • Common Lisp
  • cUrl
  • Delphi
  • Go
  • Java
  • .NET
  • Node.js
  • Objective-C
  • Perl
  • PHP
  • PHP5
  • Python
  • REALbasic
  • Ruby
