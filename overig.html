Overige API's van Facebook

Naast de Graph API heeft Facebook nog drie APIâ€™s. Hier heb je helaas niet veel aan omdat ze niet toegankelijk zijn of oud en overbodig. Daarom zullen we er hier niet veel over zeggen.

Public Feed API[1]

De Public Feed API is alleen voor een kleine groep bedrijven toegankelijk.
Deze API stuurt zelf openbare statusupdates van gebruikers over een HTTPS verbinding. Dit gebeurt in de vorm van XML objecten.

Keyword Insights API[2]

Ook de Keyword Insights API is alleen voor een kleine groep bedrijven toegankelijk.
Met deze API is het mogelijk om statistieken op te vragen over bepaalde woorden. Het opvragen moet in FQL (Facebook Query Language), wat gebaseerd is op SQL.

Chat API[3]

De Chat API is niet meer nodig omdat de functonaliteit ervan is opgenomen in de tweede versie van de Graph API. De Chat API werkt nog wel, maar zal in de toekomst uitgeschakeld worden. Daarom is het verstandiger om de Graph API te gebruiken.
Met de Chat API kunnen verschillende dingen gedaan worden:

  • authorizeren van de gebruiker
  • tekstberichten, afbeeldingen en digitale visitekaartjes sturen en ontvangen
  • meldingen sturen en ontvangen als de gebruiker of gesprekspartner aan het typen is
  • aangeven dat de gebruiker afwezig is als er een bepaalde tijd geen teken van activiteit waargenomen is
